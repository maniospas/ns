Get started with NS and NSLang

NSLang

NS lets you declare and run expressions with programmatic or precompiled C++ implementations. NSLang is a language built on top of NS by adding expressions. Here's an example of its code:

run('nslang/all.ns');	
//Print declared by NSLang
print('Hello world!');
Conditions, loops, arithmetics, and = are declared by NSLang. Conditions are scopes that are evaluate to their last expression's outcome. 
while(i=i+1;i<=10) {
	i = 0;
	print(i); //1,2,...,10
};

Scopes

Use a brackets to prevent any assignment or declaration from leaking to enclosing superscopes. Parentheses declare scopes that just run code. Do not put a semicolon after the last expression to make it the scope's returned value.

x = 0;
y = {x = x+2; x*x};
print(x); //0
print(y); //4

new references the scope object itself. Return it and access its contents with the dot notation.

x = 0;
y = {z = x+1; new};
print(y.z); //1

Expressions

Declare NS expressions with signature := implementation. The signature can have any placement of variables (in parentheses) between predicates.

add(x,y) := {x+y};
sub (z) from (x,y) := {x+y-z};
Point(x,y) := { //views x,y
  el sum := x+y; // reruns
  new //constructor
}; 
print(add(1,2)); //3
print(sub (3) from (1,2)); //0
print(Point(1,2).x); //1 
print(Point(1,2).el sum); //3

Overloading

The NS colon operator (the dot is defined by NSLang) enters a scope. It uses the scope from where it is called as a fallback variable getter. Override any formula like so.

P(x,y,z) := {
	norm sq:=x*x+y*y+z*z;
	new
};
P(x,y) := P(x,y,0);
P(x:new<0,y) := P(0,y);
p1 = P(1,1);
p2 = P(-1,1);
print(p1.norm sq); //2
print(p2.norm sq); //1

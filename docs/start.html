Get started with NS

NSLang

NS lets you declare and run expressions with programmatic or precompiled C++ implementations. NSLang is a language built on top of NS by declaring additional expressions. Here's a usage example:

run('nslang/all.ns');	
// Print declared by NSLang.
print('Hello world!');
// = declared by NSLang.
i = 0;
// Loops and arithmetics
// declared by NSLang.
// The condition scope's
// last value is checked.
while(i=i+1;i<=10) {
	print(i); // 1,2,...,10
};

Scopes

Use a brackets to prevent any assignment from leaking to enclosing superscopes. Parentheses declare scopes that just run code. Do not put a semicolon after the last expression to make it the scope's returned value.

x = 0;
y = {x = x+2; x*x};
print(x); //0
print(y); //4

new references the scope object itself. Return it and access its contents with the dot notation.

x = 0;
y = {z = x+1; new};
print(y.z); //1

Expressions

Create expressions with :=. These can have any placement of variables between predicates. Enclose arguments in parentheses.

add(x,y) := {x+y};
sub (z) from (x,y) := {x+y-z};
Point(x,y) := { //views x,y
  el sum := x+y; // reruns
  new //constructor
}; 
print(add(1,2)); //3
print(sub (3) from (1,2)); //0
print(Point(1,2).x); //1 
print(Point(1,2):el sum); //3

Self-modification

The colon enters a scope. It uses the scope from where it is called as a fallback variable getter. Override any formula like so.

system = new; // keep system
stricter = {
  (x)<(y) := {
    system: ( //in system
      (x)<((y)-(1))
    ) 
  };
  new
};
print(stricter:(1)<(2)); //modified

Welcome to NS

Programming based on scopes.

Learn a programming framework that lets you define not only methods but also new expressions it can parse. You can even interpret new languages, uncluding the accompanying NSLang! Explore a dynamic environment with assignment-safe scopes that prevent unwanted side-effects, even when using threads. Learn to instantiate objects by letting scopes return themselves but which can be re-entered later on. Plus, enjoy efficient implementations tailored for scientific computations. Enhance your toolkit with the simple yet powerful NSLang!
